shShare

Description

Factory which creates single Share❣. It allows to save it and perform actions.

Methods

  • demandReturnShare(sharedItem)

    It performs action of demand return share by invoking Parse.Cloud command

    Parameters

    • sharedItem – {sharedItem} –

      sharedItem object

  • getStateDisplay(state)

    Creates verbose version of sharedItem state See: state.

    Parameters

    • state – {string} –

      state

  • getTypeDisplay(type)

    Creates verbose version of sharedItem type See: type.

    Parameters

    • type – {string} –

      type

  • returnShare(sharedItem)

    It performs action of return share by invoking Parse.Cloud command

    Parameters

    • sharedItem – {sharedItem} –

      sharedItem object

  • saveShare(postOnFB)

    Performs full action of create SharedItem. It's invoked after click save button.

    Parameters

    • postOnFB – {Boolean} –

      if perform posting on FB

  • setDefaults()

    Sets default values when user wants add another Share

  • uploadPhoto(file)

    Upload photo to Parse

    Parameters

    • file – {Parse.File} –

      Parse.File object

Properties

  • amount

    Amount of items in share.

    Returns

    {number} –

    amount

  • date

    Date of share.

    Returns

    {date} –

    date

  • direction

    One choice from:

    TO_ME: 0,

    TO_FRIEND: 1.

  • img

    URL to share image.

    Returns

    {string} –

    img URL

  • isPublic

    if it is public share (visible for all)

    Returns

    {boolean} –

    is public

  • state

    State of share. One choice from:

    CREATED: 0,

    CONFIRMED: 1,

    RETURNED: 2,

    RETURNEDNOTCONFIRMED: 3,

    REJECTED: 4.

  • targetUser

    User who is second participant in sharing

    Returns

    {User} –

    Parse.User

  • text

    Text content of share.

    Returns

    {string} –

    Text

  • type

    One choice from:

    TIME: 0,

    THING: 1,

    PROMISE: 2.
