shUser

Description

Shared Object for keeping user's data and provides user-related methods.

Methods

  • fetchData()

    Used to fetch all user's data. Called always when shUser object is created.

  • fetchFbFriends()

    Fetches Friends List from Facebook API. When done calls shUser.fetchFriends to get proper list.

  • fetchFbUserData()

    Fetches date about current user from FacebookApi

  • fetchFriends()

    Maps Facebook Friends list fbFriendsList to Parse.User list.

  • fetchFriendsShares()

    Fetches SharedItems for contact list. It's using Cloud Code function getCommonShares.

  • fetchUserShares(loadPrevious, loadNext)

    Fetches SharedItem object's for current user's wall. Respects filters from shUser.userWallFilters.

    Parameters

    • loadPrevious – {bool} –

      notify if it should load previous shares

    • loadNext – {bool} –

      notify if it should load newest shares

  • logIn()

    Log in or Sign Up user using Facebook Account. Reload or redirect to main page when successful.

  • logOut()

    Logout current User and redirect to main page.

  • refreshParseUser()

    Checks if user has changed data on facebook and syncs with copy on parse.com.

Properties

  • fbFriendsList

    Cached list of facebook friends, where friends are facebook og:api user objects, which keeps id and username. It's used to generate friendsList. Generally FacebookUser objects are not used.

    Returns

    {Array} –

    FacebookUser

  • fbuser

    Current logged in facebook user

    Returns

    {FacebookUser} –

    FacebookUser

  • friendsList

    Cached list of friends, where friends are Parse.User objects.

    Returns

    {Array} –

    Parse users

  • friendsShares

    Cached list of SharedItem matching chosen friend's connections.

    Returns

    {Array} –

    SharedItem

  • friendsShares

    Current logged in user

    Returns

    {Parse.User} –

    Parse.User

  • loggedIn

    Returns

    {Boolean} –

  • userShares

    Cached list of SharedItem matching user filters. Must be null to distinguish if it is fetching first time.

    Returns

    {Array} –

    Filters defining user's wall SharedItems (shUser.userShares).
